Classic Pong

Created with input from a Javascript tutorial, This is the start of a work in progress as I build out my version of the classic game Pong. I couldn't resist, Pong was the first video game I have ever played. I still vividly remember the Atari console hooked up to a black & white television!
Using Javascript, HTML and CSS concepts such as Canvas API to generate the 2d objects and event listeners to keep track of ball and paddle movements to constantly re-draw canvas. The AI simply moves a paddle in relation to the ball position, but purposely lags a little to create easier player wins.
